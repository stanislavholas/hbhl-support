O aplikaci

HBHL je aplikace pro sledování požárního sportu v regionu Havlíčkův Brod. Poskytuje přehled soutěží, výsledky, žebříčky, statistiky týmů a další.

Zásady ochrany osobních údajů

Poslední aktualizace: 22. března 2026

Jaká data shromažďujeme

Aplikace HBHL neukládá žádná osobní data uživatelů. Oblíbené týmy a nastavení aplikace (jazyk, vzhled, sezóna) jsou uloženy lokálně na vašem zařízení a nejsou sdíleny s třetími stranami.

Analytika a diagnostika

Aplikace využívá služby Firebase od společnosti Google:

  • Firebase Analytics Anonymní statistiky používání aplikace. Data jsou anonymizována a nelze je přiřadit ke konkrétní osobě.
  • Firebase Crashlytics Automatické hlášení pádů aplikace pro zlepšení stability. Hlášení obsahují technické informace o zařízení, nikoliv osobní data.
  • Firebase Cloud Messaging Doručování push notifikací. Pro tento účel je generován anonymní token zařízení.

Přístup k internetu

Aplikace přistupuje k internetu pro získávání informací o soutěžích, výsledcích, žebříčcích, statistikách a bodování týmů z vlastního serveru. Dále komunikuje se službami Firebase pro výše uvedené účely.

Data třetích stran

Při použití funkce navigace na soutěž je uživatel přesměrován do externí mapové aplikace (Apple Maps, Google Maps nebo Waze). Aplikace HBHL nepředává těmto službám žádná osobní data. Sdíleny jsou pouze souřadnice cílového místa.

Vaše práva

Máte právo na přístup ke svým údajům, jejich opravu či vymazání. Pro uplatnění svých práv nebo dotazy ohledně ochrany osobních údajů nás kontaktujte na standaholas@seznam.cz.
